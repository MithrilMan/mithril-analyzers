# BGA001 File Too Long

## What It Matches

- C# files whose physical line count exceeds `mithril_analyzers.max_file_lines`

## What It Ignores

- generated files such as `*.g.cs`, `*.generated.cs`, and `*.designer.cs`
- files containing `<auto-generated`

## Configuration

```ini
[src/**/*.cs]
mithril_analyzers.max_file_lines = 800
```

## Notes

- This is a hard backstop, not a design target.
- The implementation currently counts physical lines conservatively.
